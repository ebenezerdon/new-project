How it works

  1. Drop images into the dropzone or select them using the button.
  2. Preview metadata and image orientation. Choose images to clean.
  3. Click Clean Selected and then download each cleaned image, or use Download All to get a single HTML gallery file containing your cleaned images for safe offline transfer.

Export options

You can download cleaned images individually or create a single gallery file that embeds all cleaned images for safe sharing. All cleaned files will have metadata removed client-side.
